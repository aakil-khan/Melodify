# 🎵 Melodify - Music Analytics Dashboard

## 📖 Overview

Melodify is a modern and fully responsive Music Analytics Dashboard built using React.js. It provides an intuitive interface for managing songs, artists, albums, premium plans, reviews, and analytics through a clean and reusable component-based architecture.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/aakil-khan/Melodify.git
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```

Build the project

```bash
npm run build
```
